# What is GQLite?

GQLite is a simple, no-nonsense graph querying library.

Why "Lite"?

Query parser? Gone since we can just use regular objects.

GQL schema? Bye-bye and hello to jsonschema.

Eager loading or relationships? left out to be handled by you (or by other libraries).

So we're left with a really simple concept Request exactly what we want. Validate the request arguments. That's it. By doing so - we remove any part of the system which adds unnecessary performance penalty.
